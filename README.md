# Instructor Effectiveness Modeling (EdTech Context)

Data Science assignment analyzing instructor effectiveness using batch-level course data (EDA, effectiveness scoring, ML model, evaluation, and interpretation).

## Files
- Instructor_Effectiveness_Assignment.ipynb — Full analysis notebook
- instructor_effectiveness_dataset.csv — Dataset (2000 course batches)

## How to run
1. Open the notebook in Jupyter or Google Colab
2. Make sure the CSV file is in the same folder as the notebook
3. Run all cells from top to bottom
